# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Packaging infrastructure for PyPI releases
- Single-file executable support via Shiv
- CI/CD workflows for automated testing and releases
- `--version` flag showing version, Python version, platform, and commit info
- Dynamic versioning from `hypomnemata.__version__`

## [0.9.0] - 2025-11-11

### Added
- Initial release with packaging and distribution support
- Versioning and metadata infrastructure
- Build system for sdist, wheels, and single-file executables
- GitHub Actions workflows for CI/CD
- Documentation for installation and release process

## [0.1.0] - Previous

### Added
- Core zettelkasten functionality
- ID-based linking with wiki-style syntax
- Transclusion support
- Full-text search with SQLite FTS5
- Metadata management with namespaced keys
- Alias support
- Quartz export
- CLI interface
- Asset integrity verification
- Note formatting tools
- Import/migration tools
- Link auditing
